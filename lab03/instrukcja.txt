1. Cel ćwiczenia
Celem ćwiczenia jest implementacja oraz porównanie trzech metaheurystycznych algorytmów optymalizacji:
Simulated Annealing (SA), Particle Swarm Optimization (PSO) oraz algorytmu genetycznego (GA) dla
problemu komiwojażera. Studenci pracują na klasycznym zbiorze TSPLIB berlin52, analizują jakość rozwiązań,
czas działania oraz stabilność wyników.
2. Dane wejściowe i problem badawczy
Zbiór berlin52 zawiera współrzędne 52 miast. Zadaniem jest znalezienie trasy o minimalnej długości, która
odwiedza każde miasto dokładnie raz i wraca do punktu startowego.

4. Pseudokody algorytmów
4.1. Simulated Annealing
Wejście: trasa początkowa x, temperatura początkowa T0
x_best <- x
T <- T0
dopóki T > Tmin:
    dla i = 1 do L:
        x_new <- wygeneruj_sąsiada(x)
        Δ <- koszt(x_new) - koszt(x)
        jeśli Δ < 0:
            x <- x_new
        w przeciwnym razie:
            zaakceptuj x_new z prawdopodobieństwem exp(-Δ / T)
        jeśli koszt(x) < koszt(x_best):
            x_best <- x
    T <- α * T
zwróć x_best
4.2. Algorytm genetyczny
Utwórz populację początkową P
oceń wszystkie osobniki
dopóki warunek stopu nie jest spełniony:
    wybierz rodziców
    wykonaj krzyżowanie
    wykonaj mutację
    oceń nowe osobniki
    utwórz nową populację
zwróć najlepszy osobnik
4.3. Particle Swarm Optimization dla TSP
Utwórz populację cząstek (permutacje tras)
dla każdej cząstki zapamiętaj pbest
wyznacz gbest
dopóki warunek stopu nie jest spełniony:
    dla każdej cząstki:
        wykonaj sekwencję ruchów przybliżającą cząstkę do pbest i gbest
        oceń nową trasę
        zaktualizuj pbest
    zaktualizuj gbest
zwróć gbest

5.1. Wczytanie danych berlin52
def load_tsp(filename):
    coords = []
    with open(filename, "r", encoding="utf-8") as f:
        read_coords = False
        for line in f:
            line = line.strip()
            if line == "NODE_COORD_SECTION":
                read_coords = True
                continue
            if line == "EOF":
                break
            if read_coords:
                idx, x, y = line.split()
                coords.append((float(x), float(y)))
    return coords
5.2. Długość trasy
import math
def route_length(route, coords):
    total = 0.0
    n = len(route)
    for i in range(n):
        a = coords[route[i]]
        b = coords[route[(i + 1) % n]]
        total += math.dist(a, b)
    return total
5.3. Przykład sąsiedztwa dla SA
import random
def swap_neighbour(route):
    new_route = route[:]
    i, j = random.sample(range(len(route)), 2)
    new_route[i], new_route[j] = new_route[j], new_route[i]
    return new_route

import matplotlib.pyplot as plt
def plot_route(route, coords, title="Najlepsza trasa"):
    xs = [coords[i][0] for i in route] + [coords[route[0]][0]]
    ys = [coords[i][1] for i in route] + [coords[route[0]][1]]
    plt.figure(figsize=(8, 6))
    plt.plot(xs, ys, "-o", markersize=4)
    plt.title(title)
    plt.xlabel("x")
    plt.ylabel("y")
    plt.grid(True, alpha=0.3)
    plt.show()
